Remaps the coordinate system to support radial tiling.

normalize

Rescales the X coordinate of the Segment Coordinates output into a [0, 1] range and shifts the Y coordinate into a [0, infinity) range. With it on, textures are stretched so each angular segment is filled; with it off, anything that overflows a segment is clipped away (default False)
