## Application Translations (bpy.app.translations)

### Application Translations (bpy.app.translations)

Manages language localization and message translation features in Blender, enabling scripts to integrate their own UI text into Blender's i18n system.

### Introduction

Warning

Only advanced use cases typically need this object's full interface. Regular add-ons should focus on the contexts property and the register()/unregister() functions. The pgettext() family exists for specialized scenarios (e.g. dynamic UI string assembly).

To support multiple languages in your script, build a nested dictionary:
{locale: {msg_key: msg_translation, ...}, ...} where:

- locale denotes a language code (e.g. fr), a language+country variant (e.g. pt_BR), a language+script variant (e.g. sr@latin), or a full combined form (e.g. uz_UZ@cyrilic).

- msg_key is a pair (context, original_message) - leverage provided context constants where possible.

- msg_translation holds the localized text for that language.

Then invoke bpy.app.translations.register(__name__, your_dict) from your register() function, and bpy.app.translations.unregister(__name__) from unregister().

The Manage UI translations add-on offers utilities to extract translatable strings, build the required Python dictionary, and optionally generate intermediary po files. Consult How to Translate Blender and Using i18n in Blender Code for details.

bpy.app.translations. pgettext ( msgid , msgctxt = None )

Attempt to retrieve a translated version of the supplied msgid using an optional context.

Note

The parameter order (msgid first, then optional msgctxt) differs from standard gettext, enabling single-argument calls (context falls back to `BLT_I18NCONTEXT_DEFAULT`).

Note

Generally avoid this in production add-ons—Blender's core handles translations internally. Exception: strings with format specifiers (like "File: %r"); prefer pgettext_iface()/pgettext_tip() for those.

Note

Returns msgid unchanged when i18n is disabled in the Blender build.

Parameters :

- msgid ( str ) – The text to translate.

- msgctxt ( str | None ) – The context for the translation (defaults to `BLT_I18NCONTEXT_DEFAULT`).

Returns :

The translated version (or msgid if unavailable).

Return type :

str

## Data Access (bpy.data)

### Data Access (bpy.data)

Blender's central API for programmatic data interaction.

bpy.data. data

Grants access to Blender's stored information hierarchy.

Type :

bpy.types.BlendData

`\text
import bpy

### Print all objects.
for obj in bpy.data.objects:
print(obj.name)

### Print all scene names in a list.
print(bpy.data.scenes.keys())

### Remove mesh Cube.
if \"Cube\" in bpy.data.meshes:
mesh = bpy.data.meshes[\"Cube\"]
print(\"removing mesh\", mesh)
bpy.data.meshes.remove(mesh)

### Write images into a file next to the blend.
import os
with open(os.path.splitext(bpy.data.filepath)[0] + \".txt\", 'w') as fs:
for image in bpy.data.images:
fs.write(\"{:s} {:d} x {:d}\\n\".format(image.filepath, image.size[0], image.size[1]))
