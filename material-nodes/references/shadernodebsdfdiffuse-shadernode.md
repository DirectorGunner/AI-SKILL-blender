base classes — bpy_struct, Node, NodeInternal, ShaderNode

class bpy.types. ShaderNodeBsdfHairPrincipled ( ShaderNode )

A physically grounded yet easy-to-handle shader for rendering hair and fur

model

Pick either the Chiang or the Huang model (default 'HUANG' )

- CHIANG
Chiang – Near-field hair scattering from Chiang et al. 2016, good for close-up shots but noisier when seen from afar..

- HUANG
Huang – Multi-scale hair scattering from Huang et al. 2022, works both up close and at a distance, handles elliptical cross-sections, and gives sharper highlights in forward-scattering directions..

Type :

Literal[‘CHIANG’, ‘HUANG’]

parametrization

Pick how the shader's color is parametrized (default 'COLOR' )

- ABSORPTION
Absorption Coefficient – Set the absorption coefficient "sigma_a" by hand, which is not the most intuitive route to coloring hair.

- MELANIN
Melanin Concentration – Set the melanin concentrations below for the most lifelike hair; you can look up typical concentrations for various hair types online.

- COLOR
Direct Coloring – Pick whatever color you like and let the shader estimate the absorption coefficient to match it.

Type :

Literal[‘ABSORPTION’, ‘MELANIN’, ‘COLOR’]

class bpy.types. ShaderNodeBsdfMetallic ( ShaderNode ) 

Microfacet-based metal reflection paired with a metallic Fresnel response

distribution 

Microfacet model controlling how light scatters across the rough surface (default 'BECKMANN' )

- BECKMANN
Beckmann.

- GGX
GGX.

- `MULTI_GGX`
Multiscatter GGX – GGX extended with a correction for multiple scattering, so energy is preserved and high-roughness surfaces avoid the usual darkening artifacts.

Literal[‘BECKMANN’, ‘GGX’, ‘`MULTI_GGX`’]

Physical Conductor – Conductor Fresnel computed from the complex refractive index for each color channel.

- F82
F82 Tint – A fit of the conductor Fresnel response built from the colors seen head-on and near the grazing (about 82°) direction.

Type :

Literal[‘`PHYSICAL_CONDUCTOR`’, ‘F82’]

base classes — bpy_struct, Node, NodeInternal, ShaderNode

class bpy.types. ShaderNodeBsdfPrincipled ( ShaderNode ) 

An approachable, physically grounded surface shader built on the OpenPBR model

distribution 

Microfacet model controlling how light scatters across the rough surface (default 'GGX' )

- GGX
GGX.

- `MULTI_GGX`
Multiscatter GGX – GGX extended with a correction for multiple scattering, so energy is preserved and high-roughness surfaces avoid the usual darkening artifacts.

Type :

Literal[‘GGX’, ‘`MULTI_GGX`’]

subsurface_method 

Technique used to render scattering beneath the surface (default 'BURLEY' )

- BURLEY
Christensen-Burley – A practical approximation of physically based volumetric scattering.

- `RANDOM_WALK`
Random Walk – Volumetric take on physically based scattering driven by the given scattering radius.

- `RANDOM_WALK_SKIN`
Random Walk (Skin) – Volumetric physically based scattering whose radius is tuned automatically against color textures, intended for shading skin..

Type :

Literal[‘BURLEY’, ‘`RANDOM_WALK`’, ‘`RANDOM_WALK_SKIN`’]
